\doxysubsection{Detailed Description}
Simple shell implementation that runs in both current directory and in the P\+A\+TH environment

Compontents\+:
\begin{DoxyItemize}
\item Prompt -\/ Displays shell prompt with helpful information
\item Scripting -\/ Supports scripting mode to run test cases
\item Built-\/\+In Commands -\/ Supports built-\/in commands including (cd, \#, history, !, jobs, exit)
\item Signal Handling -\/ Able to grancefully handle user pressing Ctrl+C
\item Redirection -\/ Supports file output redirection and pipe redirection
\item Background Jobs -\/ Run command in the background if command ends in \&
\item Tab Completion -\/ Autocomplete command of user if user presses tab key 
\end{DoxyItemize}
